## 1. Install Xampp and run it
### start the apache and mysql services
![image](https://github.com/fathima7674/Student-Registration-From/assets/97104966/d6cd48ed-7290-41d1-90e8-932899427be6)

## 2. Open the phpMyAdmin by using the below link
### http://localhost/phpmyadmin/
## 3. Create the DataBase by using below Query
#### CREATE TABLE students (
#### id INT AUTO_INCREMENT PRIMARY KEY,
#### name VARCHAR(50) NOT NULL,
#### email VARCHAR(50) NOT NULL,
#### course VARCHAR(50) NOT NULL
#### );
![image](https://github.com/fathima7674/Student-Registration-From/assets/97104966/dfefc0bc-3985-4494-8c14-57678dc04a16)
## 4. Create a Folder name it as StudentRegistration in c:/Xampp/htdocs/StudentRegistration
#### create index file in Xampp/htdocs/StudentRegistration/index.html and save it
#### similarly create style.css , script.js and process.php files and save it
## 5. Run the project by using below link and enter the data in the form. You will recieve a message on successful submission.
#### http://localhost/StudentRegistration/
